# react-native-managed-template

A React Native Expo starter template project with built-in navigation &amp; state management with `redux toolkit`.

## What's included?

- React Navigation v7
- Axios with interceptors
- Redux Toolkit {and peer dependencies}
- Unistyles v3 for theming

## Why should I use this template?

- Splash Screen & Authentication flow
- Pre-setup navigation and redux toolkit
- Eslint & Prettier pre-configured to work together
- Actively maintained
